Режим ученика

Вопросы появляются по одному и случайно. Чтобы вопрос исчез, нужно ответить правильно два раза. Ошибся — увидишь правильный ответ и пояснение, а вопрос вернётся позже.

Статистика сохраняется в этом браузере. Ученик может открыть статистику и сделать скриншот: закреплено, ошибки, попытки, время.

Админ-панель

Можно менять вопросы, ответы, пояснения, картинки, пароль и смотреть статистику игроков по режиму закрепления.

Картинки: можно вставить ссылку assets/q1.jpg, внешнюю ссылку https://... или загрузить файл прямо в админке. Для GitHub лучше класть картинки в папку assets и указывать путь.

На GitHub Pages сайт работает без сервера, поэтому здесь видна статистика, сохранённая в этом браузере. Для сбора со всех устройств ученики могут присылать скриншот статистики или экспорт JSON.
